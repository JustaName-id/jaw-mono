# appName

The name of your application, displayed to users during authentication and transaction signing.

**Type:** `string`
**Required:** No
**Default:** `'DApp'`

---

## Usage

```typescript
const jaw = JAW.create({
  apiKey: 'your-api-key',
  appName: 'My Awesome DApp',
});
```

---

## Description

The application name is shown to users in several places:
- Authentication popup header
- Transaction approval screens
- Permission grant dialogs
- Connected apps list in wallet

A clear, recognizable name helps users identify your application and builds trust.

---

## Best Practices

### Use Your Brand Name

```typescript
// ✅ Good - Clear brand name
const jaw = JAW.create({
  apiKey: 'your-api-key',
  appName: 'Uniswap',
});

// ❌ Bad - Generic name
const jaw = JAW.create({
  apiKey: 'your-api-key',
  appName: 'DApp',
});
```

### Keep It Concise

- Aim for under 30 characters
- Avoid long descriptions
- Use the official brand name

```typescript
// ✅ Good
appName: 'OpenSea'

// ❌ Bad - Too long
appName: 'OpenSea - The largest NFT marketplace for buying and selling digital collectibles'
```

### Avoid Special Characters

Use standard alphanumeric characters for maximum compatibility:

```typescript
// ✅ Good
appName: 'My DApp'
appName: 'Super Games'

// ⚠️ Caution - May not render properly
appName: 'My DApp™'
appName: 'Super Games 🎮'
```

  appName: 'Axie Infinity',
  appLogoUrl: 'https://axieinfinity.com/logo.png',

### Development Environments

Add environment indicator during development:

```typescript
const jaw = JAW.create({
  apiKey: process.env.API_KEY,
  appName: process.env.NODE_ENV === 'production'
    ? 'My DApp'
    : `My DApp (${process.env.NODE_ENV})`,
});
```

Result:
- Production: "My DApp"
- Development: "My DApp (development)"
- Staging: "My DApp (staging)"

## Internationalization

For multi-language support, detect user locale:

```typescript
function getAppName(locale: string) {
  const names = {
    en: 'My DApp',
    es: 'Mi DApp',
    fr: 'Mon DApp',
    ja: '私のDApp',
  };

  return names[locale] || names.en;
}

const userLocale = navigator.language.split('-')[0];

const jaw = JAW.create({
  apiKey: 'your-api-key',
  appName: getAppName(userLocale),
});
```

## Dynamic Configuration

For white-label solutions or multi-tenant apps:

```typescript
function createJAWForTenant(tenantId: string) {
  const tenants = {
    tenant1: { name: 'Brand A', logo: 'https://a.com/logo.png' },
    tenant2: { name: 'Brand B', logo: 'https://b.com/logo.png' },
  };

  const tenant = tenants[tenantId];

  return JAW.create({
    apiKey: process.env.API_KEY,
    appName: tenant.name,
    appLogoUrl: tenant.logo,
  });
}

const jaw = createJAWForTenant('tenant1');
```
