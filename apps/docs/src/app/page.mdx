# Getting Started

The JAW SDK is a TypeScript library designed for applications to create, manage, and interact with smart contract wallets on Ethereum and EVM-compatible chains.

## Overview

JAW SDK provides a comprehensive solution for integrating advanced functionalities into your application with features like:

- **Passkey Authentication** - Secure, passwordless authentication using WebAuthn
- **Gasless Transactions** - Built-in paymaster support for sponsored transactions. Users can also pay gas with ERC-20 tokens natively.
- **Atomic Batch Operations** - Execute multiple calls atomically via `wallet_sendCalls` (EIP-5792)
- **Permission System** - Grant granular spend and call permissions to third parties in a permissionless and trustless way
- **Multi-Chain Support** - Works across Ethereum and EVM-compatible chains

## Installation

Install the JAW SDK package:

```bash npm2yarn
npm install @jaw.id/core viem
```

JAW SDK uses `viem` as a peer dependency for Ethereum utilities and type definitions.

## Setup

### 1. Initialize the SDK

Create a JAW SDK instance with your configuration:

```typescript
import { JAW } from '@jaw.id/core';

const jaw = JAW.create({
  apiKey: 'your-api-key-here',
  appName: 'My DApp',
  appLogoUrl: 'https://my-dapp.com/logo.png'
});
```

The SDK returns an object with two properties:

- **`provider`** - An EIP-1193 compatible Ethereum provider for making RPC requests
- **`disconnect()`** - Method to disconnect the current session and clean up resources

### 2. Connect a Wallet

Request user authentication to connect their wallet:

```typescript
const accounts = await jaw.provider.request({
  method: 'wallet_connect'
});

console.log('Connected:', accounts[0]);
```

This opens the JAW authentication interface where users can create a new account or sign in with an existing passkey.

### 3. Make Requests

Use the provider to interact with the connected wallet:

```typescript
// Get current chain ID
const chainId = await jaw.provider.request({
  method: 'eth_chainId'
});

// Send a call bundle
const bundleId = await jaw.provider.request({
  method: 'wallet_sendCalls',
  params: [{
    version: '1.0',
    chainId: '0x1',
    from: account,
    calls: [
      {
        to: '0xContractA...',
        data: '0x...',
      },
      {
        to: '0xContractB...',
        data: '0x...',
      },
    ],
  }],
});

// Sign a message
const signature = await jaw.provider.request({
  method: 'personal_sign',
  params: ['Hello World', accounts[0]],
});
```
