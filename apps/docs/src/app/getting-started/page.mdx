# Getting Started

Learn how to install and set up `@jaw.id/core` in your project.

## Installation

Install the package using your preferred package manager:

```bash npm2yarn
npm install @jaw.id/core
```

## Basic Setup

After installation, you can import and use the package in your project:

```typescript
import { /* your imports */ } from '@jaw.id/core';
```
