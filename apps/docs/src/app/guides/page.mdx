## Available Guides

### [Subscription Payments](/guides/subscriptions)

Learn how to implement recurring subscription payments using JAW's permission system. This guide covers:

- How permissions enable subscription-like recurring payments
- Implementing subscriptions with Wagmi hooks
- Implementing subscriptions with the Core SDK
- Building a subscription service backend
- Best practices for production deployments
