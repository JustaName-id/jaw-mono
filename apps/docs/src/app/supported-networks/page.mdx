# Supported Networks

JAW supports the following networks.

---

## Mainnets

| Network | Chain ID | Chain ID (Hex)
|---------|----------|---------------
| Ethereum | `1` | `0x1`
| Optimism | `10` | `0xa`
| BNB Chain | `56` | `0x38`
| Polygon | `137` | `0x89`
| Base | `8453` | `0x2105`
| Arbitrum One | `42161` | `0xa4b1`
| Linea | `59144` | `0xe708`

---

## Testnets

Testnets are available when `preference.showTestnets` is enabled:

```typescript
const jaw = JAW.create({
  apiKey: 'your-api-key',
  preference: {
    showTestnets: true,
  },
});
```
