# Supported Networks

JAW supports the following networks.

---

## Mainnets

| Network | Chain ID
|---------|----------
| Ethereum | `1`
| Optimism | `10`
| BNB Chain | `56`
| Polygon | `137`
| Base | `8453`
| Arbitrum One | `42161`
| Linea | `59144`

---

## Testnets

Testnets are available when `preference.showTestnets` is enabled:

```typescript
const jaw = JAW.create({
  apiKey: 'your-api-key',
  preference: {
    showTestnets: true,
  },
});
```
