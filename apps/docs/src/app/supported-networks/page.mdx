# Supported Networks

JAW supports the following networks.

---

## Mainnets

| Network | Chain ID | Usage |
|---------|----------|-------|
| Ethereum | `1` | `defaultChainId: 1` |
| Optimism | `10` | `defaultChainId: 10` |
| BNB Chain | `56` | `defaultChainId: 56` |
| Polygon | `137` | `defaultChainId: 137` |
| Base | `8453` | `defaultChainId: 8453` |
| Arbitrum One | `42161` | `defaultChainId: 42161` |
| Linea | `59144` | `defaultChainId: 59144` |

---

## Testnets

Testnets are available when `preference.showTestnets` is enabled:

```typescript
const jaw = JAW.create({
  apiKey: 'your-api-key',
  preference: {
    showTestnets: true,
  },
});
```
