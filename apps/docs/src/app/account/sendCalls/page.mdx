# account.sendCalls()

Send multiple calls as a bundled user operation without waiting for receipt.

**Type:** `instance async`

---

## Signature

```typescript
async sendCalls(calls: TransactionCall[]): Promise<BundledTransactionResult>
```

---

## Parameters

### calls

Type: `TransactionCall[]`

Array of transaction calls to execute.

```typescript
interface TransactionCall {
  /** Target contract address */
  to: Address;
  /** Value to send in wei (bigint or hex string) */
  value?: bigint | string;
  /** Call data */
  data?: Hex;
}
```

---

## Returns

`Promise<BundledTransactionResult>` - The user operation ID and chain ID.

```typescript
interface BundledTransactionResult {
  /** User operation hash */
  id: Hex;
  /** Chain ID where the operation was submitted */
  chainId: number;
}
```

---

## Behavior

1. Bundles all calls into a single user operation
2. Sends the user operation to the bundler
3. **Returns immediately** with the user operation ID
4. Does not wait for transaction inclusion

For waiting until the transaction is mined, use [`sendTransaction()`](/account/sendTransaction).

### Basic Usage

```typescript
import { parseEther } from 'viem';

const { id, chainId } = await account.sendCalls([
  { to: '0x...', value: parseEther('0.1') }
]);

console.log('UserOp hash:', id);
console.log('Chain:', chainId);
```


```typescript

async function handleSend(to: Address, amount: bigint) {
  // Optimistically update UI
  addPendingTransaction({ to, amount, status: 'pending' });

    const { id } = await account.sendCalls([{ to, value: amount }]);

    // Update with actual ID
```

## sendTransaction vs sendCalls

| | `sendTransaction()` | `sendCalls()` |
|---|---|---|
| Returns | Transaction hash | User operation ID |
| Waits | Yes, until mined | No, returns immediately |
| Use case | Need confirmation | Fire and forget |
| Error handling | Includes mining errors | Only submission errors |

```typescript
// Use sendTransaction when you need the final hash
const hash = await account.sendTransaction(calls);
console.log('Mined:', hash);

// Use sendCalls when you want to continue immediately
const { id } = await account.sendCalls(calls);
console.log('Submitted:', id);
// Check status separately
```
