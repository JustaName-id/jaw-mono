# account.estimateGas()

Estimate gas for a transaction.

**Type:** `instance async`

---

## Signature

```typescript
async estimateGas(calls: TransactionCall[]): Promise<bigint>
```

---

## Parameters

### calls

Type: `TransactionCall[]`

Array of transaction calls to estimate.

```typescript
interface TransactionCall {
  /** Target contract address */
  to: Address;
  /** Value to send in wei (bigint or hex string) */
  value?: bigint | string;
  /** Call data */
  data?: Hex;
}
```

---

## Returns

`Promise<bigint>` - The estimated gas amount in wei.

---

## Behavior

Estimates the total gas required for the user operation, including:
- Call execution gas
- Verification gas
- Pre-verification gas

This estimate is used by the bundler to determine if the operation can be submitted.

### Basic Usage

```typescript
import { parseEther } from 'viem';

const gas = await account.estimateGas([
  { to: '0x...', value: parseEther('0.1') }
]);

console.log('Estimated gas:', gas.toString());
```

### Pre-flight Check

```typescript
import { formatEther } from 'viem';

async function canAffordTransaction(
  account: Account,
  calls: TransactionCall[]
): Promise<boolean> {
  try {
    const gas = await account.estimateGas(calls);
    const gasCost = await account.calculateGasCost(calls);

    console.log('Gas units:', gas.toString());
    console.log('Gas cost:', gasCost, 'ETH');

    // Check if user has enough balance
    // (implementation depends on your balance checking)
    return true;
  } catch (error) {
    console.error('Estimation failed:', error.message);
    return false;
  }
}
```

### Compare Gas for Different Operations

```typescript
import { encodeFunctionData } from 'viem';

// Estimate transfer
const transferGas = await account.estimateGas([
  {
    to: USDC_ADDRESS,
    data: encodeFunctionData({
      abi: erc20Abi,
      functionName: 'transfer',
      args: ['0x...', 1000000n],
    }),
  }
]);

// Estimate approve + swap (batched)
const swapGas = await account.estimateGas([
  {
    to: USDC_ADDRESS,
    data: encodeFunctionData({
      abi: erc20Abi,
      functionName: 'approve',
      args: [ROUTER_ADDRESS, 1000000n],
    }),
  },
  {
    to: ROUTER_ADDRESS,
    data: encodeFunctionData({
      abi: routerAbi,
      functionName: 'swap',
      args: [/* ... */],
    }),
  }
]);

console.log('Transfer gas:', transferGas.toString());
console.log('Swap gas:', swapGas.toString());
```

### Display Gas Estimate in UI

```typescript
import { formatUnits } from 'viem';

function GasEstimate({ account, calls }: Props) {
  const [gas, setGas] = useState<bigint | null>(null);
  const [loading, setLoading] = useState(false);

  useEffect(() => {
    setLoading(true);
    account.estimateGas(calls)
      .then(setGas)
      .catch(console.error)
      .finally(() => setLoading(false));
  }, [calls]);

  if (loading) return <span>Estimating...</span>;
  if (!gas) return <span>Unable to estimate</span>;

  return <span>Gas: {formatUnits(gas, 0)} units</span>;
}
```

## Error Cases

Estimation can fail for various reasons:

```typescript
try {
  const gas = await account.estimateGas(calls);
} catch (error) {
  if (error.message.includes('execution reverted')) {
    console.error('Transaction would fail');
  } else if (error.message.includes('insufficient funds')) {
    console.error('Not enough ETH');
  } else {
    console.error('Estimation failed:', error.message);
  }
}
```
